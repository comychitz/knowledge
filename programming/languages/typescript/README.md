# typescript

## creating a project
```
$ mkdir my-project
$ cd my-project
$ npm init
[follow prompts]

$ tsc --init
[follow prompts]

mkdir src out
touch ./src/index.ts

// update tsconfig.json rootDir and outDir

// add a simple start script to package.json
// for example, "start": "tsc && node ./out/index.js"
```
